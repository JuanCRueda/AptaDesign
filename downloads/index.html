Compiled Distributions
(Recommended)

Pre-compiled software distributions. These distributions are ready-to-run and reaquiere minimal installation. Therfore, they are recommended for rapid run. They don't require any external software (only the software that comes with each distribution). Download the apropiate distribution for your operative system and read the tutorials for installing and running the compiled software distributions.

Source Code

Raw Python source code for AptaDesign. Requires Python 3.5 or superior (Python 3.8 is recommended) and the installation of the modules described in requirements.txt within the download bundle. Each download bundle conatins the uncompiled Python source code optimized for each of the supported operative systems. This version allows for easy modification and cross-software integration.
Also, check the project's Github repository.
